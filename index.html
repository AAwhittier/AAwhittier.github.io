Projects

Maze Generator

Mazes are a great way to showcase algorithmic thinking skills! This maze generator, written in Kotlin, utilizes depth-first traversal and backtracking algorithms to create a randomized maze of a specified size. This allows for the creation of long pathways through the maze, the longest of which is shown in red and becomes the critical path for solving the maze! Once a maze has been generated, the software draws it to a canvas using Java's Abstract Window Toolkit. This canvas is then saved as a .PNG image to make the maze printable and savable for future use.

Conway's Game Of Life

Which cells will live and which will die? Conway's Game Of Life is a timeless programming adventure with a few simple rules. Any cell with 2-3 living neighbors will survive to the next generation, dead dells with 3 living nearby become alive, and any other cells die before the next generation. This implementation was done on the Windows Universal Platform using C# and Jframes to display the simulation.

I highly enjoyed working on this project. It was a great way to begin working with the UWP platform and further my skills as a developer. Simulations such as these show the power that programming can have in predicting and automating our lives.
